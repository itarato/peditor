Peditor
-------

Toy text editor.

[![asciicast](https://asciinema.org/a/SJonWXqbb0T6SgIQjvofFtnfv.svg)](https://asciinema.org/a/SJonWXqbb0T6SgIQjvofFtnfv)


### Compile

```bash
make clean
make
```

### Test

```bash
make clean
make runtests
```

### Use

- Open empty editor: `./pedit`
- Open file: `./pedit <FILENAME>`
- Save: `CTRL` + `s`
- Save as: `CTRL` + `o`
